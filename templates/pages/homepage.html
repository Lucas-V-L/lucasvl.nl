Welcome to LucasVL.NL!

This is the personal website of LucasVL. Here I host my personal projects, blog posts, and other cool stuff! The stylised version of this page is best viewed with javascript enabled, however it is all viewable without it enabled (you just may run into some cursor alignment issues). The source code to this site is available at My Git Server as it is written using Flask, otherwise it would quickly become a complicated mess to maintain :).

To Firefox and mobile users: I have done my absolute best to make the stylised version of this page work on your systems, however some effects have been disabled due to horrible performance issues with the displacementmap SVG filter (see Here). For some reason Safari on mobile works fine, but not on MacOS. Please use the plain version of the page if you are on MacOS
